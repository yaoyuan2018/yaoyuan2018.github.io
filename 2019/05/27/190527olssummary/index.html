1、OLS说明

最小二乘法：

 给定序列X( x1, x2, ..., xn ), y, 估计一个向量A（a0, a1, a2, ...）。令
y' = a0 + a1 * x1 + a2 * x2 + ... + an * xn，使得(y' - y) ^ 2最小，计算A。
